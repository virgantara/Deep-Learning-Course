# Tugas Kelompok: Klasifikasi Anjing vs Kucing dengan Transfer Learning

## Deskripsi Tugas:
Setiap kelompok diminta untuk membangun model klasifikasi gambar anjing dan kucing menggunakan teknik transfer learning dengan pretrained model (misalnya: VGG16, ResNet, MobileNet, dll). Kelompok diharapkan mengeksplorasi fine-tuning, augmentasi data, serta evaluasi performa model

## Output yang dikumpulkan:
1. Jupyter Notebook (.ipynb)
1. Laporan PDF (maks. 10 halaman, berisi penjelasan, kontribusi/refleksi tim, visualisasi)
1. File README.md (deskripsi cara menjalankan kode)
1. Tautan github project kelompok
